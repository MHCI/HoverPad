1. AbstractDemo
This is a demo application which runs under Android and shows how to use the framework and give a quick start into programming with the BlueMotionLib.
1.1 Additional Libraries
Make sure you include the Android IOIO and IOIO BT libraries for communication with the hardware. 

2. BlueMotionLib
This library is the elementary part to create a new application.
Make sure you mark this as libraray in your new android project. 

3. LeviTabLib
If you want to communication over wireless lan include this library into your project and mark it as such in the project properties.
