1. AbstractDemo
This is the other second part of the „AbstactDemo“ Application for Android. This second part runs on Microsoft Windows and communicates with the Android part and the hardware which is the Arduino controller box. 
1.1 Additional library
To make this project run you have to include the Newtonsoft library in C# to use JSON successfully. 

2. The CraneMotion library is build for communication with a Arduino MEGA 2560 and the correct software on it. 

3. TableServer is for communication over wireless LAN with JSON.
